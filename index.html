HTML Basics

What is an HTML Tag?

Every website you've ever seen is built with HTML tags.
Let's learn what they are — one tag at a time.

A Tag is Like a Label

Imagine you're packing boxes. You write a label on each box so everyone knows what's inside. HTML tags work the same way — they tell the browser what kind of content is inside.

<p> Opening tag
Hello, world! Content
</p> Closing tag

Quick tip: Most tags come in pairs — an opening tag <tag> and a closing tag </tag>. Notice the / in the closing tag — that's how you close it!

🃏 Meet the Tags

There are over 100 HTML tags — but don't worry! Here are the most important ones you'll use all the time.

📰
<h1>
Heading

The biggest, most important title on the page. Like a newspaper headline!

<h1>My Website</h1>
📝
<p>
Paragraph

Used for writing sentences and paragraphs — like the text in a storybook.

<p>Once upon a time...</p>
🔗
<a>
Link

Creates a clickable link that takes you to another page. "a" stands for anchor!

<a href="...">Click me!</a>
🖼️
<img>
Image

Shows a picture on the page. This tag has no closing tag — it's self-closing!

<img src="cat.png" />
🔘
<button>
Button

Creates a clickable button. You can make it do things when someone clicks it!

<button>Press me!</button>
📋
<ul> & <li>
List

Makes a bullet list. ul is the whole list; li is each item.

<ul><li>Apples</li></ul>

🤔 Did You Know?

  • 🌍 HTML was invented in 1991 by Tim Berners-Lee — the same person who invented the World Wide Web!
  • 🏷️ HTML stands for HyperText Markup Language. "Markup" means adding labels (tags) to text.
  • 📄 Every website you visit — YouTube, Google, Instagram — starts with an HTML file!
