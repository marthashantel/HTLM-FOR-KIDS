HTML Basics

What is an HTML Tag?

Every website you've ever seen is built with HTML tags.
Let's learn what they are — one tag at a time.

A Tag is Like a Label

Imagine you're packing boxes. You write a label on each box so everyone knows what's inside. HTML tags work the same way — they tell the browser what kind of content is inside.

<p> Opening tag
Hello, world! Content
</p> Closing tag

Quick tip: Most tags come in pairs — an opening tag <tag> and a closing tag </tag>. Notice the / in the closing tag — that's how you close it!
